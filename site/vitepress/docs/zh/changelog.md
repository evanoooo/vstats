## [1.0.0] - 2024-01-01

### 新增
- 🚀 首次发布
- 通过 WebSocket 实时服务器监控
- 多服务器管理面板
- CPU、内存、磁盘、网络指标
- NVIDIA GPU 监控支持
- JWT 认证
- Docker 部署支持
- Linux/macOS/Windows Agent 支持
- 脚本安装便捷部署
- SQLite 数据库存储数据
- 精美的玻璃拟态 UI 设计
- 多主题选项
- 响应式 Web 界面

### 功能
- **面板**：现代化的服务器监控 Web 界面
- **Agent**：轻量级监控探针，支持 Linux、macOS、Windows
- **API**：REST API 用于集成
- **WebSocket**：实时指标更新
- **认证**：安全的 JWT 认证
- **部署**：Docker 和脚本安装
