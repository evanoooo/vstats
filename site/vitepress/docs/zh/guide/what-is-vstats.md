# 什么是 vStats？

vStats 是一个极简美观的服务器监控面板。使用 Go 和 React 构建，提供毫秒级延迟的实时系统指标。

## 特性

### ⚡ 实时监控
- WebSocket 驱动的实时数据推送
- 毫秒级延迟
- 实时图表和图形

### 🖥️ 多服务器管理
- 单一面板监控多台服务器
- 统一的服务器列表视图
- 独立的服务器详情页面

### 📊 全面指标
- **CPU**: 使用率、核心数、频率、每核统计
- **内存**: 总量、已用、可用、交换分区
- **磁盘**: 多磁盘支持、I/O 统计
- **网络**: 带宽、接口、总流量
- **GPU**: NVIDIA GPU 支持（可选）
- **系统**: 运行时间、负载平均、操作系统信息

### 🎨 现代 UI
- 玻璃拟态设计
- 流畅动画
- 多主题选项
- 响应式布局

### 🔐 安全
- 基于 JWT 的认证
- 仅管理员操作
- 安全的 WebSocket 连接

### 🚀 易于部署
- Docker 一键部署
- 脚本安装
- Systemd 服务管理

## 架构

vStats 由两个主要组件组成：

```
┌─────────────────────────────────────────────────────────────┐
│                    vStats Dashboard                         │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐          │
│  │   Web UI    │  │  REST API   │  │  WebSocket  │          │
│  │   (React)   │  │    (Gin)    │  │  (Gorilla)  │          │
│  └─────────────┘  └─────────────┘  └─────────────┘          │
│                          │                                   │
│              ┌───────────┴───────────┐                       │
│              │      Go Backend       │                       │
│              │    + SQLite DB        │                       │
│              └───────────────────────┘                       │
└─────────────────────────────────────────────────────────────┘
                           │ WebSocket
           ┌───────────────┼───────────────┐
           ▼               ▼               ▼
    ┌─────────────┐ ┌─────────────┐ ┌─────────────┐
    │   Agent 1   │ │   Agent 2   │ │   Agent N   │
    │   (Linux)   │ │  (Windows)  │ │   (macOS)   │
    └─────────────┘ └─────────────┘ └─────────────┘
```

### Dashboard（服务端）
主要组件，负责：
- 提供 Web UI
- 提供 REST API
- 管理 WebSocket 连接
- 在 SQLite 中存储服务器数据

### Agent（探针）
轻量级探针，负责：
- 收集系统指标
- 通过 WebSocket 发送数据
- 作为系统服务运行
- 支持 Linux、Windows、macOS

## 技术栈

| 组件 | 技术 |
|------|------|
| 后端 | Go (Gin, Gorilla WebSocket) |
| 前端 | React, TypeScript, Tailwind CSS |
| 数据库 | SQLite |
| 部署 | Docker, Systemd |

## 系统要求

### 服务端
- Linux、macOS 或 Windows
- 最低 512MB 内存
- 100MB 磁盘空间
- 端口 3001（可配置）

### Agent
- Linux、macOS 或 Windows
- 64MB 内存
- 可访问服务端网络
