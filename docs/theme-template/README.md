# My Custom Theme for VStats

A custom theme for [VStats](https://github.com/zsai001/vstats) server monitoring dashboard.

## Installation

1. Go to **Settings** → **Themes** → **Install Theme**
2. Enter: `your-username/my-custom-theme`
3. Click **Install**

Or install a specific version:

```
your-username/my-custom-theme@v1.0.0
```

## Customization

Edit `theme.css` to customize colors and styles.
